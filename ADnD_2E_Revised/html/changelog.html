Announcements

Changes in

  • Updated suggested Tome of Magic spheres to Clerics, Druids, Paladins, and Rangers based on the Priest's Spell Compendium
  • Multi-class THAC0 supported with macros
  • Added Concealed and secret door search buttons
  • Added Item saving throws table
  • Updated Traits, Special Talents and Disadvantages to use roll template for output
  • Fixed Traits rating roll to correctly add modifier
  • Added missing label for Traits, Special Talents, Disadvantages, and Nonweapon proficiencies checks
  • Added actual classes to dropdown for each of the class fields. This is a move towards requiring the correct class is placed in the correct field
